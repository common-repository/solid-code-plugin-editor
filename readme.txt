== Description ==

An enhanced plugin editor with a better tree view of the plugin files and folders and some extra functionality.

#### WARNING

Changing plugin files may harm your WordPress Site installation, please do not install this plugin unless you know exactly what you are doing.
Make sure you backup all your files before installing this plugin

#### Features and highlights:

* View any plugin files in a tree view structure (including sub directories)
* View all files (not just PHP and CSS)
* Edit the following file extensions: PHP, CSS, JS, XML, HTML, HTM, TXT
* Download a ZIP backup of the whole plugin
* Download individual files

== Installation ==

To install this plugin please use the "install" feature from the WordPress site.
